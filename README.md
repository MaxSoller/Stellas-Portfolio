# Stella Victor — Industrial Design Portfolio

A clean, minimal portfolio website built with HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Live Site

🔗 [https://maxsoller.github.io/Stellas-Portfolio/](https://maxsoller.github.io/Stellas-Portfolio/)

## Customization

- **Content**: Edit `index.html` to update text, project descriptions, and links.
- **Images**: Replace placeholder divs with `<img>` tags pointing to images in the `images/` folder.
- **Styling**: Tweak CSS custom properties in `:root` at the top of `styles.css`.
- **Contact Form**: Replace `your-form-id` in the form action with your [Formspree](https://formspree.io) endpoint, or switch to a `mailto:` link.

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Google Fonts (Inter)
- GitHub Pages for hosting
